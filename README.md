# Welcome to the My-Gas-Detector wiki!

Simple cross platform application to display ( Air Quality, CARBON MONOXIDE, ... ) received from esp8266 using MQ-9, MQ-135, DHT11 .

# hardware Setup



## Wire

| DHT11 | ESP8266 |
|-------|---------|
| GND   | GND     |
| 3V3   | 3V3     |
| Data  | D7      |


| 4051 | ESP8266 |
|------|---------|
| GND  | GND     |
| 3V3  | 3V3     |
| X    | A0      |
